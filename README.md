# Python Workouts 🐍

A comprehensive collection of production-ready Python scripts and exercises for learning, practice, and skill development across 18+ domains and difficulty levels. Perfect for beginners to advanced developers, interview preparation, and daily coding practice.

## 📚 Overview

This repository contains structured Python exercises and examples covering:
- **Basics**: String manipulation, number operations, and fundamental concepts
- **Data Structures**: List, dictionary, and advanced data structure operations
- **Algorithms**: Sorting, searching, dynamic programming, graph algorithms, and tree traversals
- **File Handling**: Reading/writing files, JSON, CSV operations
- **Web Scraping**: Basic web scraping with BeautifulSoup
- **API Interactions**: REST API client implementation
- **Data Analysis**: Pandas for data manipulation and analysis
- **Automation**: File and system automation scripts
- **OOP**: Object-oriented programming concepts and patterns
- **Testing**: Unit testing with unittest framework
- **Database Operations**: SQLite, SQLAlchemy ORM, and connection pooling
- **Web Development**: Flask, FastAPI, REST APIs, and template rendering
- **Machine Learning**: Scikit-learn basics, data preprocessing, and model training
- **Data Visualization**: Matplotlib, Seaborn, and Plotly
- **Concurrency**: Threading, async/await, and multiprocessing
- **DevOps Tools**: Docker, CI/CD, and cloud API interactions
- **Security**: Encryption, secure coding practices, and authentication
- **Performance**: Profiling, memory management, and caching strategies
- **Advanced Testing**: pytest, mock objects, and integration testing

## 🚀 Getting Started

### Prerequisites

- Python 3.7 or higher
- pip (Python package installer)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/nellaivijay/python-workouts.git
cd python-workouts
```

2. Install required dependencies:
```bash
pip install -r requirements.txt
```

Each script can be run independently:

```bash

python basics/string_manipulation.py
python basics/number_operations.py

# Data Structures
python data_structures/list_operations.py
python data_structures/dictionary_operations.py

# Algorithms
python algorithms/sorting_algorithms.py
python algorithms/search_algorithms.py
python algorithms/dynamic_programming.py
python algorithms/graph_algorithms.py
python algorithms/tree_traversals.py

python file_handling/file_operations.py

python web_scraping/basic_scraping.py

# API Interactions
python api_interactions/rest_api_client.py

# Data Analysis
python data_analysis/pandas_basics.py

# Automation
python automation/file_automation.py

# OOP
python oop/classes_examples.py

python testing/unit_tests.py

# Database Operations
python database_operations/sqlite_examples.py
```
